# Tree操作
## ツリー階層変更
### 共通 Key Binding

###### 短縮
`Meta(Alt) + 方向キー`

###### Spacemacs メニュー階層
| `SPC` | `m` | `S` |
|:--|:--|:--|
| Spacemancs root | +major-mode-cmd | +subtree |

### Key Bindings
| Operation | Abbr Key | SPC Key | func | Description |
|:--|:--|:--|:--|:--|:--|
| 階層下げ（→） |  `M-→` |`SPC m S l`| org-demote-subtree |  |
| 階層上げ（←） |  `M-←` |`SPC m S h`| org-promote-subtree |  |
| １つ前へ（↑） |  `M-↑`|`SPC m S k`| org-move-subtree-up |  |
| １つ後へ（↓） |  `M-↓`|`SPC m S j`| org-move-subtree-up |  |
