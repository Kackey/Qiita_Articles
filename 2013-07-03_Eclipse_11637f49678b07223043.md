- 4.2などは、プロジェクトのプロパティメニューから
- 4.3では、プロジェクトのプロパティメニューに表示がない。
-- とりあえず.projectファイルを直接書き換えでJavaプロジェクトには変換できる

# 4.2以前

## Operation
> "Properties..." -> "Project Facets"

## Reference
- [Eclipseでプロジェクトのタイプを後から変更 ： eclipse, java | メモリークラフト](http://www.memorycraft.jp/2008/11/eclipse.html)

# 4.3 Kepler他

## 直接書き換え

```:before
<?xml version="1.0" encoding="UTF-8"?>
<projectDescription>
	<name>hogehoge project</name>
	<comment></comment>
	<projects>
	</projects>
	<buildSpec>
	</buildSpec>
	<natures>
	</natures>
</projectDescription>
```

```:after
<?xml version="1.0" encoding="UTF-8"?>
<projectDescription>
	<name>hogehoge project</name>
	<comment></comment>
	<projects>
	</projects>
	<buildSpec>
		<buildCommand>
			<name>org.eclipse.jdt.core.javabuilder</name>
			<arguments>
			</arguments>
		</buildCommand>
	</buildSpec>
	<natures>
		<nature>org.eclipse.jdt.core.javanature</nature>
	</natures>
</projectDescription>
```
