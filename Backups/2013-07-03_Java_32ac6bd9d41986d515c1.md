# ポイントメモ
- propertiesファイルのエンコーディングは、"ISO-8859-1"
- Eclipseの場合、設定ですでに"ISO-8859-1"で指定されている。
- 通常のフリーのエディタでは２バイト文字部分を読めないので注意。
