# PC Install

##### [WebStorm 7 — Everything you need, at your fingertips, without compromise](http://www.jetbrains.com/webstorm/whatsnew/)
- JSでの開発に特化。syntax hilightの対応が広い。
- 細かなツールが便利。debuggerとか、REST Clientとか。
- テスティングフレームワークのサポートが広い。jsTestDriver, Karma, mochaがサポート。Karma使うと、ファイル更新入るたびにテスト結果がGUI表示されて快適。istanbulもサポートしたらしい。
- nodeもサポート。
- 有料。

##### [IntelliJ IDEA — The Best Java and Polyglot IDE](http://www.jetbrains.com/idea/)
- WebStormの上位版。Java, Ruby, PythonもOK。
- 有料。けっこう高い。

###### [Aptana](http://www.aptana.com/)
- eclipseベース。
- 無料

# Cloud
###### [Cloud9 IDE | Your code anywhere, anytime](https://c9.io/)
- 本格的な統合開発環境。
- RailsやPHPでサーバサイドも一緒に開発して、そのまま公開できる。
- Herokuにも連携してたりで、そのままデプロイできたりする。
- WordPressも開発できるようになったらしい。
  - [クラウド最強エディタ「Cloud9」が、WordPressもブラウザのみでインスト・開発・公開が出来て無料！ | APPGIGA!!（アプギガ）](http://plus.appgiga.jp/masatolan/2014/01/23/50260/)

###### [jsdo.it - Share JavaScript, HTML5 and CSS](http://jsdo.it/)

###### [Create a new fiddle - JSFiddle](http://jsfiddle.net/)
- 本格的な開発環境ではない。
- コードサンプルを置いたり、スニペット置いたり。
- ブログに組み込んでサンプルコードを出せたりとか。書籍のサンプルコード置き場に使われたりもする。
